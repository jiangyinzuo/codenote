vim:ft=help
*codenote.txt*

manually set *g:coderepo_dir* and *g:noterepo_dir*

>vim
	" ends with /
	let g:coderepo_dir = [["duckdb/duckdb", "/path/to/duckdb/"]]
	let g:noterepo_dir = "/path/to/note/"
<

Commands~

	:OpenNoteRepo
		Open the note repo and initialize link signs.

	:OpenCodeRepo
		Open configured code repos in tabs and initialize link signs.

	:CodenoteRefreshLinks
		Rescan Markdown notes and refresh link signs.

Keymaps~

	<leader>ny
		Yank the current line, or the current visual selection, as:
			pathline
			code block
		Then jump to the note tab.

	<leader>nl
		Yank only the current cursor line as a pathline.
		Then jump to the note tab.

	<leader><C-]>
		Jump between code and note.

	<leader>np
		Preview the matching note snippet for the current code location.

Pathline format~

	repo_name:path/to/file:start-end
